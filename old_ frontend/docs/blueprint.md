# **App Name**: ResumeMatcher

## Core Features:

- User Authentication: Secure signup and login functionality for users.
- Resume Upload: Allow users to upload their resumes.
- Job Description Upload: Enable users to upload job descriptions.
- Resume Parsing: Automatically parse uploaded resumes to extract key information.
- Job Matching: tool; Utilize generative AI to match resumes to job descriptions based on skills, experience, and keywords. Consider if, when and how to incorporate new data.
- Matching Score: Provide a score indicating the relevancy of the resume to the job description.
- Service Pages: Different pages showcasing the model services.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to inspire trust and professionalism.
- Background color: Light gray (#F0F4F8) for a clean and modern feel.
- Accent color: Teal (#26A69A) for highlights and calls to action, indicating growth and opportunity.
- Body text: 'Inter' sans-serif for readability.
- Headline font: 'Space Grotesk' sans-serif to bring tech, computer related aesthetic
- Use clear, professional icons related to job searching and resume analysis.
- Maintain a clean and organized layout with clear sections for resume uploads, job postings, and matching results.
